title: Tickle
---

Displays the time remaining until your next resting tick.

## Settings

```
tickle <command> [arg1 [arg2 [...]]]
```

|  Command  |  Arguments  |  Default  |  Description  |
|:---|:---|:---|:---|
| **pos** |  `<x>` <br> `<y>`  | 0 <br> 0 | Sets the position of the text. |
